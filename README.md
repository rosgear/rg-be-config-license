## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-license/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Лицензия»

Управление лицензионным ключом редакции веб‑приложения RosGear: получение, активация, проверка статуса.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Конфигурация»](https://github.com/rosgear/rg-be-config).

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-config-license
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-config-license": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-config-license/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-license/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «License»

Managing the license key for the RosGear web application edition: obtaining, activating, and checking the status.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«Configuration» module](https://github.com/rosgear/rg-be-config).

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-config-license
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-config-license": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-config-license/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
